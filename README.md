# ESP32 LED Blink Project (Rust)

## Overview
This project demonstrates a basic embedded system using Rust and the ESP32 microcontroller platform inside the Wokwi simulator.

The system initializes the ESP32 hardware and controls an LED using GPIO output logic. The LED state is toggled continuously inside a loop with timed delays.

The purpose of the project is to demonstrate embedded Rust programming concepts including:
- GPIO control
- Hardware initialization
- Delays and looping
- Cargo dependency management
- ESP32 simulation with Wokwi
- Embedded systems debugging

## Running the Project

1. Open the project in Visual Studio Code
2. Install the Wokwi extension
3. Build the project
4. Open the Wokwi simulator
5. Run the simulation
